# sentinel

[![Build Status](http://10.177.233.77:8080/buildStatus/icon?job=sentinel)](http://10.177.233.77:8080/job/sentinel/)

> The mutation testing tool for the meta-shift project

## How to build

    $ cmake .
    $ make all


## How to test

    $ cmake -DENABLE_TEST=ON .
    $ make all test coverage
